# 학생 진단 리포트

## 개요

전국 수학 학원 학생들의 학습 상태를 AI가 자동 분석하여 개인화된 진단 리포트를 생성하는 핵심 기능입니다. 학부모에게 이해하기 쉬운 형태로 학생의 취약점과 추천 학습 경로를 제공합니다.

## 핵심 가치

- **정확한 진단**: AI 기반 취약점 분석으로 90% 이상 정확도
- **맞춤 피드백**: 학생별 개인화된 학습 추천
- **학부모 신뢰**: 명확하고 이해하기 쉬운 리포트 형식

## 기대 효과

1. 학부모 만족도 향상 → 재등록률 증가
2. 교사 업무 부담 감소 → 교육 품질 집중
3. 학원 차별화 → 시장 경쟁력 강화
